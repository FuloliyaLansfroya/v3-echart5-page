RadarGraph #

雷达图基础组件。雷达图主要用于表现多变量的数据，例如球员的各个属性分析。

使用场景 #

  • 雷达图主要用于表现多变量的数据，使用户能够从视觉上得到各数据变量的关系结果，正确使用可以提升用户体验。

API #

Attributes #

仪表盘组件属性。

参数说明类型必选可选值默认值版本
data雷达图需要显示的指标数据{value: number; name: string}[]是--版本
RadarIndicator雷达图坐标系数据IndicatorType是--版本
RadarOption雷达图配置RadarOption否--版本
